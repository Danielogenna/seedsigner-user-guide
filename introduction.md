# SeedSigner User Guide

## Introduction
SeedSigner is an open-source project that enables users to build an affordable, air-gapped Bitcoin signing device using off-the-shelf hardware components.

### Key Features

* Open-source design
* DIY assembly
* Air-gapped transaction signing
* QR code-based interaction
* Support for multiple seed generation methods
* Fully offline operation

### Why Use SeedSigner?
- Affordable and accessible
- No internet connection required, improving security
- Easy to assemble and operate

### Important Safety Tips
- Always verify the authenticity of the SeedSigner software.
- Never expose your seed phrase to anyone.
- Use multiple backups for your seed.
